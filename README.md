# ShoppingCartProductwithDB

**Features**

Display a list of products from a database.
Add products to the shopping cart.
Remove products from the shopping cart.
Session-based cart management.

**Technologies Used**

C#
ASP.NET Web Forms
ADO.NET
SQL Server

**Project Structure**

AppProps: Contains classes representing the application properties, such as Cart and MyProduct.
DAL: Contains Data Access Layer classes for interacting with the database.
BLL: Would contain Business Logic Layer classes, if implemented.
ShoppingCartProductwithDB: Contains the web forms, controls, and master pages.

**Usage**

**Viewing Products**

On the home page, a list of products from the database is displayed.

**Adding Products to Cart**

Click the "Add to Cart" button next to a product to add it to the shopping cart.
The cart is managed using session variables.

**Removing Products from Cart**

In the cart view, click the "Remove" button next to a product to remove it from the shopping cart.
